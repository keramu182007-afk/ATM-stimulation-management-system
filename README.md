# 🏧 ATM Simulation System

## 📌 Description
The ATM Simulation System is a console-based application that simulates the working of a real-world Automated Teller Machine (ATM). It allows users to perform basic banking operations such as checking balance, depositing money, withdrawing cash, and viewing transaction history.

This project is developed to understand core programming concepts like data structures, file handling, and user authentication.

---

## 🚀 Features
- 🔐 Secure PIN-based login
- 💰 Balance inquiry
- 💵 Cash deposit
- 🏦 Cash withdrawal
- 📜 Transaction history
- ⚠️ Error handling for invalid inputs
